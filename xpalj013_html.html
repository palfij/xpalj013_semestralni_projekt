Úvod

Pro tento semestrální projekt jsem si vybral neexistující Online Rádio. Zvolil jsem si ho z toho důvodu, že sám rád poslouchám muziku a mám rád možnost využití poslechu online.

XML soubor zde slouží k nastavení online aplikace, práv jednotlivých uživatelů podle jejich úrovně (například platící zákazníci budou mít více možností než uživatel co poslouchá zdarma), a také k nastavení jednotlivých sekcí (stylů) rádia.

Metodika

Při tvorbě tohoto programu jsem používal program NetBeans IDE 8.2 a internetový prohlížeč Google Chrome.

Popis elementů

Název elementu Popis elementu Schématické omezení
radio_nastaveni Zaobaluje nastavení aplikace. Obal - bez omezení.
nazev Určuje název rádia. String omezení.
nastaveni-radia Zaobaluje obecné nastavení. Obal - bez omezení.
protokol Určuje jaký protokol bude použit pro zobrazení stránky. Protokol http, nebo https.
charset Určuje jaký charset bude používán. Charset utf-8, nebo windows-1258.
jazyk Určuje národostní jazyk aplikace. Jazyk český, nebo americká angličtina.
javascript Určuje zda bude povolený javascript. Hodnoty 0,1.
nastaveni-doplnku Zaobaluje nastavení použitých doplňků. Obal - bez omezení.
komentare Určuje zda bude povolen doplněk komentářů. Hodnoty 0,1.
vyhledavani Určuje zda bude povolen doplněk vyhledávání. Hodnoty 0,1.
omezeni-pristupu Blacklist IP adres se zakázaným přístupem. String omezení.
kontakt-admin Kontaktní e-mail na správce. String omezení.
druhy-uzivatelu Zaobaluje nastavení pravomocí uživatelů. Obal - bez omezení.
role Zaobaluje jednotlivé pravomoce a identifikuje je podle atributu Unazev. Obal - bez omezení.
reklamy Určuje zda uživatel bude mít při poslechu zobrazené reklamy. Hodnoty 0,1.
volba-skladeb Určuje zda uživatel může měnit skladby. Hodnoty 0,1.
uprava-popisu Určuje zda uživatel může měnit popis skladeb. Hodnoty 0,1.
pridat-favorite Určuje zda uživatel může přidat skladbu do jeho favoritů. Hodnoty 0,1.
stupen Určuje míru jeho možností manipulace/oprávnění. Hodnoty 0,1,2,3.
styl Zaobaluje jednotlivé styly. Obal - bez omezení.
popis Popisuje danou sekci. Popis o 0 až 80 charakterech.
uzivatele Zaobaluje nastavení uživatelských oprávnění pro danou sekci. Obal - bez omezení.
user Zaobaluje nastavení jednodtlivých uživatelů v této sekci, identifikuje uživatele podle id. Obal - bez omezení.
jmeno Určuje jméno uživatele. String omezení.
prijmeni Určuje příjmení uživatele. String omezení.
login Určuje login uživatele. String omezení.
e-mail Určuje e-mail uživatele. String omezení.
premium-dnu Určuje kolik uživateli zbývá prémium dnů. Bytové omezení.
uroven-uctu Určuje uživatelovu míru jeho možností manipulace/oprávnění. Hodnoty 0,1,2,3.

Popis atributů

Název atributu Popis atributu
Unazev Určuje název druhu uživatele.
id Určuje identfikátor uživatele.
Snazev Určuje název stylu (sekce).

Popis transformace

Transformace probíhá pomocí cyklu for-each a JSON, který vybere uživatele a jejich údaje, kterým zbývá 10 a více prémium dní, seřadí je vzestupně podle příjmení a pomocí testu zjistí, pokud se jedná o poslední údaj, případně doplní čárku.
